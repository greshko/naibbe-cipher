sherigza
sttligzat t rizgar od
hwatchkan oogzt gazintle
tligz sherigza groohw
litlin goog razitlttlain
grang kan hiztll t
azkat t hwagan t rizar
tligz hizargoo n hwara
lingt oora zitlin ziltou
kerinat kerinoot diz oo
ligtlig ootin ootin hwigen kerinoot
kan sherizga zir koonigan hwin
hwagan sherizga tltloon oogriz kerinoot
t raltain ooztll kan karasz
too nkiztlooz iz koningtl oo
liztlara sherizga azat oora tizlt
aret zir hwara kerinat tizlt arat
zir hwara kerinoot izkarat tliz titlin
girhwiz tagzig gizhwara ootin ootin
sherizga kan ziltoo gerin gerin tlogz gerhwiz
sherizga rizga ootin griztlin griza

tloot tloot

sttligzat t rizgar oo hwatchkarr rizktliz
oogzt tloot tloot kan tloot tloot grain grizk
lizkat rizkat tlerain kiz kan tizkan hwiz
hwigu rizt tloozkin tloot tloot gizak tlerigain
giztat tlirak tloot tloot toot tin toot hwkjen
tloot tloot oogriz tlerain koogan kan litglig
gizkan rizkan gwerigeran koozkooz kerinoot
kernaat tltltlinkenkizoonkizak t kan oo nzk tloot
tloot t kerinoot diz glizglizik lookoor kerinoot

kizbak

sttligzat trizgar
oo hwatchkum rizktliz goog
rizt kizbak rekizan tlizk izak
iz hwigen kizkiz rizook tleriska gr
grit tltltlinkenkizoon zir kuratl kizook rizk
hwarain keroot rizkin kan gizhwark kihwkim
tltltlinkenkizoon zihwoon t ziz t zigrd tlerain mizhwain koogin kan
gerhwikan tlik tergin tlik tlootz tlakerain zizhwar
t gark iztlaru kizbak azat oora hwigen zizkanat ootin
ootin zir kizook ooztl tlikanut hwara n tltlain goog
hwazan razitlttlain oogriz zirhar zirhazar kishreker tlik
tak zizek rizkat azik tlinken ootin azat hwin hwagan kizbuk
tizket tltltlinkenkizoon izk kan kizkak oo zinkizkin kuegan
zir karkakitl lizk t rizharkara oora tzil oo kahw grain
lizk hizgtlgtliga oo zir kanoot goog tlerain zizirik hwizgar
lizhwoon kan n kanagan kerootgagiz rizak kizbak
